Comparing Revisions in the Log View

To view the changes made to a file in a particular revision:

  1. Select the file of interest by clicking the filename in the revision’s change list, expanding the change list as necessary.

  2. Select View > Show Differences (key equivalent ⌘⏎) or click the Show Differences button in the log view’s toolbar:

    Show Differences

    Alternatively, you can also double-click a file in the revision’s change list to open the compare view.

  3. A compare view will be shown displaying the changes made in the selected revision.

Press Cancel (key equivalent Escape ⎋) to return to the log view from the compare view.
