CORNERSTONE 4 BUY SUPPORT RELEASE NOTES

Adding an Existing Working Copy

If you already have Subversion working copies on your Mac then you need to add them to the source list in Cornerstone.

You can either drag the working copy folder from the Finder to Cornerstone’s source list or:

  1. Select File > Add Working Copy… (or click the button located at the bottom of the source list and select Add Working Copy…)

    You can also click the Add Working Copy... button in the toolbar.

  2. Use the displayed panel to select a working copy folder.
  3. Click the Add button to add the working copy to the source list.

Cornerstone will automatically detect the version of your working copy and use the correct Subversion version when accessing it.
